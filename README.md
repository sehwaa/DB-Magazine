## Technology Stack
- Web Framework : Flask
- Language : Python

## Fuction
- 로그인/로그아웃
- 회원탈퇴
- 금주의 게임왕(테트리스, 리그오브레전드 랭킹)
- 김기자의 칼럼 컬렉션(DB반 소속 김기자의 칼럼 리스트)
- 점심식사 추천(항상 점심식사 고민하는 친구들을 위한 점심식사 추천)
- 출결관리(출석, 지각, 결석 관리)
- DB반 프로젝트 및 기타 일정 관리
- 자유게시판(익명으로 대화 나눌 수 있는 게시판)
- 오늘의 몸풀기 동영상 시청(테드 강연)
